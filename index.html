Console Project

To set up visula studio you need to download it first. You then need to make a folder and open it in visula studio. Name it "index.html". To see how your code looks, save the project and click on go live at the bottom right corner.

To go to the console screen, you need to right click on the website, and then click on inspect. On the top, you can go the console screen to see if you ocde to show things on the console screen worked.
